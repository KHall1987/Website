Package Label Verification

Project information

  • Category: Computer Vision
  • Client: West Fraser
  • Project date: October 8, 2020 to February 3, 2021
  • Project URL: www.example.com

In an effort to automate the package outfeed section of the blue ridge planer mill a system was designed to adhear labels to the sides and backs of packages which meets West Fraser's standards for labeling and identification. Due to the industrial enviroment that this was in a solution that didn't require direct interaction from site staff was preffered. So a novel label application and verification system was developed to adhere and check labels being attached to outgoing packages. The verification process checked for correct package information including, product, length, dimension, UPC, and colour code. In addition the label verification system checked to make sure that the labels were adhered correctly and that the text was legible and that ink levels were adequate.

A cognex system was selected for this application, along with suitable lens to capture the area of interest with minimal distortion. Shop tests verified that the test labels could be read relibly with the camera system. When on site issues with the non-uniformity of the package sides lead to less than desirable results initially. however through adjustment of acceptable tolerances a solution was found that satisfied the requirments for the project.
